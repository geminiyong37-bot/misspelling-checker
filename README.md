# AI 문서 맞춤법 검사기 (Misspelling Checker)

오타 찾기에 지친 분들을 위해 개발된 **AI 기반 문서 맞춤법 검사기**입니다.  
HWP, HWPX, PDF 등 다양한 문서에서 오타와 문법 오류를 AI가 대신 찾아 엑셀로 정리해 드립니다.

## 주요 기능

- **다양한 파일 형식 지원**: HWP, HWPX, PDF, DOC, DOCX, TXT 파일을 지원합니다.
- **AI 오타 검증**: Gemini, OpenAI, Anthropic 등 최신 AI를 활용하여 정확도 높은 오타 검사를 수행합니다.
- **사용자 친화적 UI**: 드래그 앤 드롭 방식으로 누구나 쉽게 파일을 업로드하고 검사할 수 있습니다.
- **엑셀 결과 저장**: 확인된 오타 리스트를 엑셀 파일(.xlsx)로 깔끔하게 내보낼 수 있습니다.
- **하이라이트 기능**: 중요한 오타라고 판단되는 부분을 색깔로 강조하여 가독성을 높였습니다.

## 사용 방법

1. **설치 파일 실행**: `AI_Word_Speller_Setup.exe`를 실행하여 프로그램을 설치합니다.
2. **파일 업로드**: 프로그램의 상단 박스 안에 검사할 파일을 드래그 앤 드롭(Drag & Drop) 합니다.
3. **API 키 설정**: 처음 실행 시 OpenAI, Anthropic 또는 Google Gemini의 API 키를 입력합니다. (가성비 좋은 모델을 기본으로 사용하여 토큰 소모가 적습니다.)
4. **검사 수행**: 업로드 즉시 AI가 분석을 시작하며, 진행 상황을 화면에서 확인할 수 있습니다.
5. **결과 다운로드**: **[결과 다운로드]** 버튼을 눌러 확인된 오타 리스트를 엑셀로 저장합니다.

## 주의 사항

- **이미지 기반 PDF**: 글자가 텍스트가 아닌 이미지로 된 PDF는 인식이 되지 않아 오류가 발생할 수 있습니다.
- **AI의 한계**: AI는 가끔 실수를 할 수 있습니다. 중요한 문서는 최종적으로 사람이 한 번 더 확인하는 것을 권장합니다.
- **API 보안**: 가급적 개인 유료 API 키를 사용하시는 것이 보안과 속도 면에서 유리합니다. (무료 API는 학습용으로 데이터가 활용될 수 있습니다.)
