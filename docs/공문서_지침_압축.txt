[공문서 작성 지침 압축 요약본 (맞춤법 및 띄어쓰기 우선 적용 규정)]

1. 연월일 및 날짜 표기:
- 월, 일 표기 시 '0'은 붙이지 않음. (예: 2022.03.09. -> 2022. 3. 9.)
- 기간 표기는 물결표(~) 또는 붙임표(-)를 사용하고 앞뒤를 붙여 씀. (예: 2022. 5.~7. / 10:00~12:00)
- 날짜 뒤 '기준', '자', '부' 표기: '2022. 10. 28. 기준' (띄움), '2022. 9. 1. 자로' (띄움), '2022. 9. 1.부로' (붙임)

2. 시간 표기:
- 24시각제 사용, 시·분 글자 대신 쌍점(:) 사용. (예: 오후 3시 20분 -> 15:20)

3. 숫자 표기:
- 30여 명, 3억 원~4억 원 (수사 범위 표시 시 수사 생략 불가)
- 이상/이하(초과/미만) 짝을 맞춰 사용. 앞말과 띄어 씀. (예: 30% 이상, 10명 이하)

4. 항목 기호 및 특수 기호 표기:
- 항목 기호(□, ○, -, · 등)와 내용 사이는 반드시 1칸 띄움. (예: - 목적: / ○ 일시: )
- 쌍점(:)은 앞말에 붙이고 뒷말과 띄움. (예: 일시: 2022. 1. 1.)
- 인용할 법령명은 낫표(「 」)를 사용. (예: 「국가계약법」)

5. 주요 띄어쓰기 규정:
- 의존 명사는 띄어 씀: 공모 건, 신청 건, 두 가지, 서울과 부산 간, 연체 여부, 학생 수, 학급 수
- 단위를 나타내는 명사는 띄어 씀: 50만 명, 10만 톤, 23만 개, 50여 회, 5천억 원, 296억 달러
- 접미사/접두사는 붙여 씀: 50여 명, 내일쯤, 20%가량, 100만 원짜리, 이틀간, 총300대, 본제도
- '내(內), 외(外), 전(前), 후(後), 시(時), 중(中), 말(末), 초(初)'은 원칙적으로 띄어 씀.
  예: 학교 내, 용도 외, 경기 전, 방과 후, 계약 시, 회의 중, 4월 말, 연초
  (단, 부재중, 한밤중, 비상시, 평상시, 무의식중 등 사전에 한 단어로 등재된 합성어는 예외)
- '간(間)': 관계·사이를 나타낼 때 띄어 씀 (서울과 부산 간, 기관 간), 기간을 나타낼 때 접미사로 붙여 씀 (이틀간, 2개월간)
- '지체 없이', '실효성 있게', '차질 없이'는 띄어 씀. ('있다', '없다'는 일부 굳어진 말 제외 앞말과 띄어 씀)
- '우리 청', '우리 교육청' (단, 우리나라, 우리말은 붙임)
- '총-': 수사 앞에서는 관형사로 띄어 씀 (총 300대), 명사 앞에서는 접두사로 붙여 씀 (총면적, 총인원)
- '전(全/前)': 관형사이므로 뒷말과 띄어 씀 (전 직원, 전 교장)
- '약'과 '-여'는 동시에 쓰지 않음: 약 30명 또는 30여 명 (약 30여 명 X)
- '대비'는 앞말과 띄어 씀 (전년 대비)
- '상(上)'이 추상적 공간의 뜻일 때는 접미사로 붙여 씀 (기본계획상, 영수증상)

6. 문장 부호 규칙:
- 가운뎃점(·): 대등한 말을 줄여 쓸 때 사용. 단, 사전에 한 단어로 등재된 경우 붙여 씀 (국공립, 공사립, 시도)
  예: 제·개정(○), 제개정(X) / 정·현원(○), 정현원(X)
- 물결표(~): 기간·범위 표시 시 앞뒤 붙여 씀 (2022. 5.~7., 6~9급, 20~30%)
- 붙임표(-): 차례로 이어지는 내용 열거 시 사용 (도움닫기-도약-착지)
- 소괄호(): 보충 내용 앞말에 붙여 씀. 조사는 괄호 앞 말 기준으로 선택 (계약서(안)를)
- 인용 시 낫표(「」) 사용: 법령명 인용 시 (「국가계약법」)
- 준말(줄임말) 사용 시 원래 용어를 먼저 쓰고 괄호 안에 준말 기재 (지방자치단체(이하 지자체))

7. 순화어(대체어) 목록:
- 통보한 → 알린 / 개소 → 곳 / 익일/익월/익년 → 다음 날/다음 달/다음 해
- 유관기관 → 관계 기관 / 실시하다 → 하다 (점검을 실시하다 → 점검하다)
- 시건장치 → 잠금장치 / 징구 서류 → 요구 서류 / 체크리스트 → 점검표
- 타 학교 → 다른 학교 / 금년/금번/금회 → 올해/이번
- 프린트 → 출력물(인쇄물) / 프린터 → 인쇄기
- 당초 → 기존 / 소관 → 담당 / 상기 → 위의 / 향후 → 앞으로
- 동 ( ), 본 ( ) → 이 ( ) / 당해 → 해당 / 공란 → 빈칸 / 식순 → 차례
- 내역서 → 명세서 / 내역 → 내용 / 10매 → 10장 / 퇴청 → 퇴근
- 탑재하다 → 올리다, 게시하다 / 홈페이지 → 누리집 / 매뉴얼 → 지침서, 안내서
- 가이드라인 → 지침, 방침 / 인프라 → 기반 시설 / 모니터링 → 점검, 실태 조사
- 기(旣) → 이미 / 필히 → 반드시 / 면밀히 → 자세히 / 강구하다 → 마련하다
- 간주하다 → 보다, 여기다 / 제고하다 → 높이다 / 자문을 받다 → 자문하다
- 만전을 기하여 → 허술함이 없도록 하여 / 기일을 엄수하여 → 꼭 지켜

8. 번역투·상투적 표현 교정:
- '~에 대하여/대해서' → '~에게', '~을(를)' (직원들에 대하여 → 직원들에게)
- '~에 있어서' → '~에서', '~할 때' (선거에 있어서 → 선거에서)
- '~로 인하여' → '~로' (부패행위로 인하여 → 부패행위로)
- 불필요한 사동 표현 제거: 구체화시켜→구체화해, 확산시킬→확산할, 소개시켜→소개하여
- '건립 중에 있으며' → '건립하고 있으며' / '진행 중에 있습니다' → '진행하고 있습니다'
- '기간 동안' → '기간' 또는 '동안' (의미 중복이므로 하나만 사용)
- '아래와 같이' 다음에 다시 '아래' 반복 금지 (의미 중복)
- '적의 조치 요망' → '적절히 조치해 주시기 바랍니다' (권위적 표현 금지)

9. 어휘 혼용 주의:
- 결재(문서 승인) vs 결제(대금 지급)
- 운용(자본·법·제도) vs 운영(조직·기구·사업체)
- 보전(환경·생태계) vs 보존(유물·영토)
- 이상/이하 vs 초과/미만: 반드시 짝을 맞춰 사용 (이상↔미만, 이하↔초과)
- 참고(하나의 자료를 도움으로 활용) vs 참조(서로 다른 자료를 비교·대조)
- 주최(전체 기획) vs 주관(실질적 업무 수행) / 둘 다일 때: 주최·주관

10. 공문서 구조 및 표기 규칙:
- 항목 체계: 1. → 가. → 1) → 가) → (1) → (가) → ① 순서. 필요 시 □ ○ - · 등 특수 기호 사용 가능
- 항목이 하나만 있으면 항목 기호를 부여하지 않음
- 항목 기호와 내용 사이 1칸 띄움. 두 줄 이상이면 둘째 줄부터 항목 내용 첫 글자에 맞춰 정렬 (원칙)
- "끝" 표시: 본문 마지막 글자에서 한 글자 띄고 "끝" 표시. 표로 끝날 경우 표 아래 왼쪽에서 한 글자 띄고 "끝"
